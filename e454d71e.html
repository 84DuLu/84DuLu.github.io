  3. 如何得到一个二进制补码整数的相反数？（反转每一位，然后加 1）

八进制

  1. 八进制数 451 在 C 中写作 0451。
  2. 每个八进制位对应 3 个二进制位。

十六进制

  1. 十六进制数 A3F 在 C 中写作 0xA3F。
  2. 在 C 语言中，A~F 既可用小写也可用大写。
  3. 每个十六进制位都对应一个 4 位的二进制数（即 4 个二进制位），那么两个十六进制位恰好对应一个 8 位字节。第 1 个十六进制表示前 4 位，第 2 个十六进制表示后 4 位。因此，十六进制很适合表示字节值。

C按位运算符

按位逻辑运算符

  1. 二进制反码或按位取反：~

    一元运算符 ~ 把 1 变为 0，把 0 变为 1。

    ~(10011010) // 表达式

    (01100101) // 结果值

    假设 num 是 int 类型的数，~num 不会改变 num 的值，就像 3 * num 不会改变 num 的值一样。如果要把 num 的值改为 ~num，可以用 num = ~num; 。

  2. 按位与：&

    二元运算符 & 通过逐位比较两个运算对象，生成一个新值。对于每个位，只有两个运算对象中相应的位都为 1 时，结果才为 1。

    (10010011) & (00111101) // 表达式

    (00010001) // 结果值

    C 有一个按位与和复制结合的运算符：&=

    val &= 0377; 即为 val = val & 0377;

  3. 按位或：|

    二元运算符 | ，通过逐对比较两个运算对象，生成一个新值。对于每个位，如果两个运算对象中相应的位位 1，结果就为 1。

    (10010011) | (00111101) // 表达式

    (10111111) // 结果值

    C 有一个按位或和赋值结合的运算符：|=

  4. 按位异或：^

    二元运算符 ^ 逐位比较两个对象。对于每个位，如果两个运算对象中相应的位一个为 1（但不是两个都为 1），结果为 1。

    (10010011) ^ (00111101) // 表达式

    (10101110) // 结果值

    C 有一个按位异或和赋值结合的运算符：^=

用法

  1. 掩码（mask）

    flags = flags & MASK;

  2. 打开位

    我们想打开一号位，其他位保持不变（MASK 只有 1 号位为 1）

    flags = flags | MASK;

  3. 关闭位

    我们想关闭一号位，其他位保持不变（MASK 只有 1 号位为 1）

    flags = flags & ~MASK;

  4. 切换位

    切换位指的是打开已关闭的位，或关闭已打开的位。

    flags = flags ^ MASK;

    flags 中与 MASK 为 1 的位相对应的位都会被切换，MASK 为 0 的位相对应的位不变。

  5. 检查位的值

    检查 flags 中 1 号位是否被设为 1 。

    if ((flags & MASK) == MASK)

    puts("Wow!");

    由于按位运算符的优先级比 == 低，所以必须在 flags & MASK 周围加上圆括号。

移位运算符

  1. 左移：<<

    左移运算符 << 将其左侧运算对象每一位的值向左移动其右侧运算对象指定的位数。左侧运算对象移出左末端位的值丢失，用 0 填充空出的位置。

    (10001010) << 2 // 表达式

    (00101000) // 结果值

    该操作产生了一个新的位值，但是不改变其运算对象。例如，假设 stonk 为 1，那么 stonk << 2 为 4，但是 stonk 本身不变。可以用左移赋值运算符（<<=） 来改变量的值。例如，stonk <<= 2;。

  2. 右移：>>

    右移运算符 >> 将其左侧运算对象每一位的值向右移动其右侧运算对象指定的位数。左侧运算对象移出右末端位的值丢失。对于无符号类型，用 0 填充空出的位置，对于有符号类型，其结果取决于机器。

    下面是无符号值的例子：

    (10001010) >> 2 // 表达式，无符号值

    (00100010) // 结果值

    右移运算符（>>=）将其左侧的变量向右移动指定数量的位数。
